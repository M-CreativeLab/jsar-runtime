✨ 将 Web 技术带入元宇宙

JSAR 是一个革命性的浏览器引擎库，专为下一代计算时代（元宇宙）而设计。它使您能够使用现代 Web 技术（HTML5、CSS3、TypeScript）创建沉浸式 Web 内容，并将其无缝集成到主流 3D 引擎中，如 Unity、UnrealEngine 和 RealityKit。

🎯 3D 优先设计

与传统浏览器不同，JSAR 从底层针对 3D 和空间计算进行了优化

🔧 易于集成

支持多种渲染后端和平台集成

📱 跨平台兼容

支持 Android 和 macOS 平台

⚡ 高性能

内置 ECS 实现 GPU 驱动的元素渲染器

🌐 Web 标准兼容

支持 WebXR、WebGL、Web API 和其他现代 Web 标准

支持的 URL 格式

与经典 Web 浏览器一样，JSAR 不仅支持 HTML 文档，还支持多种格式：

格式 状态 推荐用例
HTML 🔄 部分支持 浏览经典 Web 页面
GLTF ✅ 支持 显示 3D 模型
JPEG/PNG ✅ 支持 查看图像
SVG ❌ 未开始 显示 SVG 图像
